# Contributing

Thanks for contributing.

## Scope

This project is intentionally small.
Changes should preserve that quality.

Good contributions:

- visual refinements that improve the site without bloating it
- accessibility improvements
- mobile layout improvements
- SEO or performance improvements
- small copy edits that fit the tone

Please avoid:

- adding frameworks unless there is a strong reason
- adding trackers beyond the current privacy-conscious setup
- turning the site into a multi-page marketing project

## Development

Serve the repository with any static file server and test on both desktop and mobile sizes.

## Pull Requests

- keep PRs focused
- describe what changed and why
- include screenshots for UI changes when useful
- do not mix unrelated changes

## Style

- prefer plain HTML, CSS, and JavaScript
- keep the design intentional and minimal
- preserve the established visual tone unless the change explicitly explores a new direction
